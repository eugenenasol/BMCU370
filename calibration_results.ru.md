# Результаты калибровки и проверенные алгоритмы BMCU

В данном документе зафиксированы параметры и последовательности команд, которые обеспечили стабильную работу системы (LOAD/UNLOAD) на текущей конфигурации железа.

## 1. Проверенные параметры системы (STATUS)
Эти параметры обеспечивают уверенное движение моторов и точную реакцию на давление.

| Параметр | Значение | Описание |
| :--- | :--- | :--- |
| **p_gain** | 5000.0 | Агрессивность ПИД давления |
| **p_min_pwm** | 180.0 | Минимальное усилие для старта мотора |
| **p_boost_thr** | 0.020 | Порог срабатывания рывка (Boost) |
| **p_boost_pwm** | 350.0 | Мощность рывка |
| **p_boost_time** | 50 | Длительность рывка (мс) |
| **p_tol** | 0.015 | Допуск (мертвая зона) давления |
| **p_offset** | -0.070 | Офсет для режима OVERFLOW |
| **m_p** | 25.0 | Пропорциональный коэффициент движения |
| **m_i** | 80.0 | Интегральный коэффициент движения |
| **m_pwm_zero** | 200 | Минимальный ШИМ для обычного движения |

## 2. Алгоритм загрузки (LOAD)
Последовательность действий для подачи филамента из BMU в экструдер.

1.  **Поиск экструдера (FTE)**:
    `{"cmd": "FEED_TO_EXTRUDER", "lane": 3, "speed": 30, "max_mm": 250.0, "pressure_thr": 0.2}`
    *(Примечание: Скорость 30 и порог 0.2 с усиленными ПИДами движения обеспечивают надежную парковку даже на тяжелых каналах).*
2.  **Захват экструдером**:
    Команда экструдеру принтера: `G1 E10 F300` (подтянуть на 10 мм).
3.  **Включение сопровождения**:
    `{"cmd": "SET_AUTO_FEED", "lane": 3, "enable": true, "overflow": false}`
4.  **Протяжка/Прочистка**:
    Команда экструдеру принтера на нужную длину (например, `G1 E100 F600`).

---

## 3. Алгоритм выгрузки (UNLOAD)
Последовательность действий для возврата филамента из экструдера в BMU.

1.  **Отрезка/Подготовка**: (Cut filament в Прюше/Бамбу стиле).
2.  **Вывод из хотэнда**:
    Команда экструдеру принтера: `G1 E-120 F1200`.
3.  **Возврат в BMU**:
    `{"cmd": "MOVE", "axis": "3", "dist_mm": -170, "speed": 50}`
4.  **Отключение контроля**:
    `{"cmd": "SET_AUTO_FEED", "lane": 3, "enable": false, "overflow": false}`

---

## 4. Схема полярности (Hardware Standard)
*   **Логический Плюс (+)** = Движение Вперед (к экструдеру).
*   **Логический Минус (-)** = Движение Назад (к катушке).
*   **Энкодер**: Падение значений при движении вперед.
*   **Инверсия в HAL**: `MOTOR_INVERT_CHx = true`.
